Html

iframe

iframe 元素会创建包含另外一个文档的内联框架（即行内框架）。

场景：在vue项目中，需要外链一个百度网页，但是跳转到百度还需要返回项目;
比如html5小项目，这个时候需要在页面上加入返回按钮，然后通过iframe标签引入百度

js部分

src 链接到的地址URl
frameborder 是否显示边框（0无边框 1有边框）
需要使同层元素不被覆盖 加style：z-index = -1
